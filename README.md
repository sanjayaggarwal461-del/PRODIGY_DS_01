# PRODIGY_DS_01

## Task 01: Data Visualization

This project is part of the Prodigy InfoTech Data Science Internship.

### Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Dataset
Titanic Dataset

### Visualization
Bar Chart showing the Gender Distribution of Titanic Passengers.

### Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

### Observation
The dataset contains significantly more male passengers than female passengers.
